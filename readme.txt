main file:
    project.py
to generate a new terrain with a texture run:
    project.py -n -t

separate features(see report.pdf):
    world.py
    dino.py
    geizer.py

Controls:
(for a QWERTY keyboard)
    W : move the dinosaur forward
    A : turn left    
    D : turn right
    Q, E : rotate the camera
    X : switch walk/run
    R : roar
    ENTER : attack
    F : eat
    
For debug purposes:
    V : switch between camera modes trackball/following
    I,K,J,L,U,N : translate the camera (absolute coordinates) (only in trackball mode)
    SPACE : reset time
